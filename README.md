# Wedding (React + TypeScript + Vite)

Dự án React cơ bản được khởi tạo bằng Vite.

## Yêu cầu

- Node.js (khuyến nghị bản LTS)

## Chạy local

```bash
npm install
npm run dev
```

Mở theo URL mà terminal in ra (thường là `http://localhost:5173`).

## Build production

```bash
npm run build
npm run preview
```

## Cấu trúc chính

- `src/App.tsx`: UI/logic chính
- `src/index.css`: style nền + base theme
- `src/App.css`: layout cho trang chính
